# ✨ Overview

Hospital Management System is a **modern healthcare management dashboard** developed using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

The application is designed to simplify day-to-day hospital operations by providing an elegant dashboard for managing patients, doctors, appointments, pharmacy, laboratory reports, billing, rooms, departments, staff, and analytics—all from a clean, responsive interface.

This project is suitable for:

- 🎓 College Projects
- 💼 Portfolio Showcase
- 🚀 Resume Projects
- 💻 Web Development Practice
- 🏥 Hospital Demonstrations
